Convert Infix to Postfix

Algorithm to convert infix to postfix:

  1. Scan the infix expression from left to right.
  2. If the scanned character is an operand, output it.
  3. If the scanned character is an operator and the stack is empty, push it to the stack.
  4. If the scanned character is an operator and the stack is not empty, compare its precedence with the operator on top of the stack. If it has higher precedence, push it to the stack; otherwise, pop operators from the stack and add to the output until the stack is empty or an operator with a lower precedence is found.
  5. If the scanned character is '(', push it to the stack.
  6. If the scanned character is ')', pop from the stack and add to the output until '(' is found.
  7. Repeat steps 2-6 until the infix expression is fully scanned.
  8. Pop all remaining operators from the stack and add to the output.
